TalkNight timer
===============

Small timer tool, that firmly notifies presenter. Requires python-virtualenv package.

Setup
-----
* install ```apt-get install mplayer2 python-virtualenv```
* invoke ```make initenv```

Usage
-----
```
timer.sh --first 1200 --other 600 600
```
